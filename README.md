# Klaytn — Scope Whale Tracker

Real-time detector for **single transactions moving 50 million+ KLAY** on Klaytn (Kakao’s enterprise blockchain).

Klaytn is used by millions in Korea via KakaoTalk.  
Retail doesn’t move this much.  
Only three types of entities do:

- Kakao itself
- Korean exchanges (Upbit, Bithumb)
- Government-level partners

## Run

```bash
python klaytn_scope_whale.py
